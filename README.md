# Rails Event App
- is a Ruby on Rails 7 application for browsing and managing events with friendly slugs. Visitors can view upcoming, past, free, or recent events.
- Signed-in users can register, like/unlike events, and see who else liked or registered.
- Admins can create, edit, and delete events; users manage their own profiles.
- Events support categories, capacity tracking, pricing (free vs paid), and basic auth via email/password.
